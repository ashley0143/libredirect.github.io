Keyboard Shortcuts

LibRedirect offers the following Keyboard shortcuts for easier access

  • Shift+Alt+L - To Switch the instance you are currently in
  • Shift+Alt+C - Copies the original link. Ex: Copies the original Twitter link when you are in Nitter website
  • Shift+Alt+O - Redirects to original Site from the frontend
  • Shift+Alt+R - Redirects from Original Site to the frontend.

Some shortcuts may or may not work in certain browsers such as Microsoft Edge.

Exceptions

Exceptions as the name says for itself will exclude certain sites from being redirected. When you add a site to the exceptions list it will not redirect as it is excluded from redirection.

This exceptions list is an allowlist NOT a blocklist.

Defined Instances

You should be able to see instances such as Cloudflare

Cloudflare: indicates that an instances is using Cloudflare as it's CDN. Cloudflare basically decrypts the data between you and the instance to compress it to send it faster, a CDN basically. In other words, Cloudflare can see your data in Plain text even with HTTPS. So, use these instances carefully.

Setting LibRedirect as a Search Engine in Chromium

Type in your address bar: chrome://settings/searchEngines. Then on Site search click Add. Then type:
Search engine: LibRedirect
Shortcut: libredirect
URL with %s in place of query: http://search.libredirect.invalid/?q=%s
